ゼロ・ロジック

すべてのパワーを使い切り、最短ルートでゴールを目指せ!
数字の計算（ゴールまでに数字を0にする）と歩数制限（ターン数）を組み合わせた、シンプルながら奥の深い思考型パズルゲームです。

ゲーム概要
プレイヤーはグリッド上の数字タイルを回収・消費しながらゴールを目指します。
ただゴールに行くだけでなく、盤面にある数字をすべて消し、かつ自分のパワーをピッタリ0にする必要があります。

ルール
・青いタイル (+):ポイントが数字の分増えます。
・赤いタイル (-):数字の分ポイントを消費します。現在のポイントがタイルの数値より低いと通れません。
・歩数制限: 1マス動くごとに減少します。0になるとゲームオーバーです。
・クリア条件:
    1. 盤面の数字タイルをすべて消す
    2. ポイントをピッタリ「0」にする
    3. ゴールに到達する

遊び方
1. 以下のリンクを開くと遊ぶことができます。https://miyajimas2525-star.github.io/Zero-Logic/
2. キーボードの矢印キーで移動します。
3. Rキーでリセット、Zキーで一歩戻ることができます。
