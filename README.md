ゼロ・ロジック

すべてのパワーを使い切り、最短ルートでゴールを目指せ!
数字の計算と歩数制限を組み合わせた、シンプルながら奥の深い思考型パズルゲームです。

ゲーム概要
プレイヤーはグリッド上の数字タイルを回収・消費しながらゴールを目指します。
ただゴールに行くだけでなく、盤面にある数字をすべて消し、かつ自分のパワーをピッタリ0にする必要があります。

ルール
・青いタイル (+):ポイントをチャージします。
・赤いタイル (-):ポイントを消費します。現在のパワーがタイルの数値より低いと通れません。
・歩数制限 (Steps): 1マス動くごとに減少します。0になるとゲームオーバーです。
・クリア条件:
    1. 盤面の数字タイルをすべて消す
    2. パワーをピッタリ「0」にする
    3. ゴールに到達する

遊び方
1. index.htmlをブラウザで開きます。
2. キーボードの矢印キーで移動します。
3. Rキーでリセット、Zキーで一歩戻ることができます。

テクニカル
使用言語:HTML5,CSS3,JavaScript(Vanilla JS)

主な機能:
    ・ステージ管理システム
    ・一歩戻る機能
    ・見やすいUI設計
